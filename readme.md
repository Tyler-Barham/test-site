# Website Test

---

## Setup
    $ cd Frontend/
    $ npm install

---

## Running Locally

The frontend and backend need to be run in their own terminals

### Backend
    $ cd Backend/
    $ php -S 127.0.0.1:8090 -t ./

### Frontend
    $ cd Frontend/
    $ npm start

---
Navigate to [http://localhost:4200/](http://localhost:4200)
